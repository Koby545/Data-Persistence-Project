# 🧱 Bricks Game (Unity)

A classic brick breaker-style game built with Unity, featuring a persistent high score system and clean UI flow.

## 🎮 Overview

This project is a complete arcade-style game where the player controls a paddle to break bricks and achieve the highest possible score.
The game includes a full menu system, score tracking, and persistent leaderboard functionality.

## ✨ Features

* 🧠 Persistent High Score System (saved locally)
* 📊 Leaderboard with player names and scores
* 🎮 Smooth gameplay mechanics (paddle, ball physics, collisions)
* 🧱 Brick destruction system
* 🧭 Main menu and in-game UI
* 🔁 Restart and game flow management

## 🛠 Technical Highlights

* Implemented a **save/load system** using PlayerPrefs (or local storage)
* Designed a **score tracking and leaderboard system**
* Built modular game logic for paddle, ball, and brick interactions
* Managed UI flow between menu, gameplay, and game over states
* Handled collision detection and physics interactions

## 🧩 Architecture

The project is structured into separate components:

* Game Manager – controls game state and flow
* UI Manager – handles menus and score display
* Score System – manages score and persistence
* Player Controller – paddle movement
* Ball Controller – movement and collision logic

## 🚀 How to Play

* Move the paddle to bounce the ball
* Break all bricks to score points
* Try to reach the highest score and enter the leaderboard
